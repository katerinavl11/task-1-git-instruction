# task-1-git-instruction
Шпаргалка по работе с Git
Как работать с репозиториями?
1. Локальный репозиторий
Чтобы создать локальный репозиторий, необходимо:
- завести папку через терминал с помощью mkdir
- находясь в этой папке ввести команду git init 
- по итогу внутри созданной вами папки появится папка с расширением .git, которая будет хранить в себе служебную информацию 
2. Удаленный репозиторий
Вариант 1. Если вы уже создали локальный репозиторий способом, описанным выше, но хотите публиковать проекты и изменения в удаленном репозитории, то есть несколько вариантов:
- создайте репозиторий на GitHub 
- введите в терминале команду git remote add origin git@github.com:никнейм/имя-репозитория.git , чтобы связать локальный репозиторий с удаленным 
- создайте хотя бы один коммит, например добавьте файл README.md следующим образом: 
echo "# My Awesome Project" >> README.md
git add README.md
git commit -m "Initial commit"
- отправьте изменения на GitHub с помощью команды git push -u origin main
Вариант 27. Если вы еще не создавали локальный репозиторий:
- создайте репозиторий на GitHub с тремя обязательными файлами
- скопируйте SSH ссылку на ваш репозиторий (пример: git@github.com:никнейм/имя-репозитория.git)
- перейдите в нужную вам папку и введите в терминале команду git clone git@github.com:никнейм/имя-репозитория.git , которая склонирует репозиторий к вам на ПК
Вы великолепны :)
